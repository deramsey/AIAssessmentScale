AI Assessment Scale

This is an interactive version of the basic AI assessment scale originally developed by Perkins, Furze, Roe, and MacVaugh. This scale was designed to be a practical framework to guide the appropriate and ethical use of generative AI in assessment design, empowering educators to make purposeful, evidence-based decisions.

Use the slider below to set the level of the scale and the "Copy to Clipboard" button to copy the associated content for easy pasting in documents or online courses.

For more information about the scale, visit the original documentation: AI Assessment Scale
